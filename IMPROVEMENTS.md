# Улучшения MVC приложения

## 📋 Обзор изменений

Данный документ описывает все улучшения, внесенные в MVC приложение для повышения безопасности, архитектуры и поддерживаемости кода.

## 🔒 Безопасность

### 1. Исправление SQL Injection уязвимости
**Файл:** `app/core/Model.php`

- Использование параметризированных запросов вместо прямой конкатенации значений
- Валидация имен колонок перед использованием в запросах
- Добавлена проверка на допустимые символы в именах колонок

```php
// Было (уязвимо):
$query->where($query->expr()->eq('t.' . $column, $value));

// Стало (безопасно):
$query->andWhere('t.' . $column . ' = :' . $paramName)
      ->setParameter($paramName, $value);
```

### 2. Защита от XSS атак
**Файлы:** `app/core/View.php`, контроллеры

- Автоматическое экранирование всех строковых переменных в шаблонах
- Добавлен метод `escapeVars()` для рекурсивного экранирования данных
- Явное использование `htmlspecialchars()` при сохранении данных в контроллерах

### 3. Улучшение CSRF защиты
**Файлы:** `app/controllers/AdminController.php`, `app/controllers/MainController.php`

- Регенерация CSRF токена после успешного входа
- Удаление CSRF токена при выходе из системы
- Проверка CSRF токена во всех POST-запросах

### 4. Валидация данных
**Новый файл:** `src/Validator/Validator.php`

Создан универсальный класс валидатора с методами:
- `required()` - проверка обязательных полей
- `email()` - валидация e-mail
- `minLength()` / `maxLength()` - проверка длины строки
- `passwordStrength()` - проверка сложности пароля

## 🏗️ Архитектура

### 1. Обновление Doctrine ORM
**Файл:** `app/core/Db.php`

- Переход с аннотаций на атрибуты PHP 8+ (`createAttributeMetadataConfiguration`)
- Вынос `$isDevMode` в конфигурацию через `env('DEV_MODE')`
- Добавлен charset `utf8mb4` для поддержки emoji и других символов

### 2. Обработка ошибок
**Новый файл:** `src/Exception/SecurityException.php`

- Создан специальный класс исключений для ошибок безопасности
- Замена `die()` на выбрасывание исключений

## 📝 Примеры использования

### Валидация в контроллере

```php
use src\Validator\Validator;

$validator = new Validator();
$validator->required($post['name'], 'name', 'Поле обязательно');
$validator->email($post['email'], 'email', 'Неверный email');

if (!$validator->isValid()) {
    foreach ($validator->getErrors() as $error) {
        Alerts::addFlash(Alerts::TYPE_WARNING, $error);
    }
}
```

### Безопасная пагинация

```php
// Теперь безопасно передавать параметры от пользователя
$pagination = Tasks::pagination(
    ['status' => 1], 
    ['created_at' => 'DESC'], 
    $page
);
```
